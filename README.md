# Personal Portfolio Website

## Project Name

Personal Portfolio Website

## Short Project Description

This is a personal portfolio website created using HTML and CSS. The website showcases information about me, my software development journey, my goals, and my reflections as an aspiring software developer.

## Purpose of the Website

The purpose of this website is to present my background, skills, learning journey, and future goals while demonstrating my understanding of web development concepts.

## Features of the Project

* Home page with welcome section
* About Me page with profile picture and personal information
* Journey page with images and descriptions of my learning journey
* Goals page with image cards showing my future goals
* Reflection page
* Contact page with a contact form
* Navigation bar linking all pages
* Responsive design
* Table displaying skills and information
* Footer on every page

## HTML Concepts Used

* HTML document structure
* DOCTYPE declaration
* Head and body elements
* Meta tags
* Headings
* Paragraphs
* Links
* Images with alt text
* Lists (ordered and unordered)
* Tables
* Forms
* Labels and input fields
* Buttons
* Semantic elements (header, nav, main, section, footer)

## CSS Concepts Used

* External stylesheet
* Selectors
* Classes and IDs
* Colours and backgrounds
* Fonts and typography
* Margins and padding
* Borders and border radius
* Flexbox
* Hover effects
* Box shadows
* Responsive design using media queries

## How to View the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open the index.html file in your web browser.
4. Navigate through the website using the navigation bar.

## Challenges Faced

One of the main challenges was designing a website that was both visually appealing and easy to navigate. I also faced challenges with styling pages consistently, organizing content, and ensuring that all links and images worked correctly. Creating responsive layouts and positioning elements properly required problem-solving and attention to detail.

## What I Learned

Through this project, I learned how to build a complete multi-page website using HTML and CSS. I improved my understanding of website structure, navigation, styling, forms, tables, images, and responsive design. I also strengthened my problem-solving and debugging skills.
