Project Reflection

Creating this portfolio website helped me understand how HTML and CSS work together. I learned how to structure webpages, create navigation menus, add images, and style content.

The challenges that I faced were organizing the pages and making sure all buttons worked correctly and finding the perfect color coordination. Through practice and persistence, I improved my coding skills and gained confidence in building websites. This project was a great learning experience and has motivated me to explore web development just like software development because they go hand in hand.
